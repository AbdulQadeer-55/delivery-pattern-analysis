# Delivery Pattern Analysis

## Project Overview
This tool analyzes daily fuel delivery logs (Excel/CSV) to identify customer delivery patterns, frequency (Weekly, Bi-Weekly, Monthly), and preferred days of the week. It also forecasts the next expected delivery date.

## Structure
- `data/`: Place raw CSV/Excel delivery logs here.
- `output/`: Generated reports will be saved here.
- `src/`: Source code for analysis.
